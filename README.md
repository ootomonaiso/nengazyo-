# 🐴 年賀状 2026 - Happy New Year!

エンジニア＆オタク向けのインタラクティブ年賀状サイトです。

## ✨ 特徴

- 🖥️ **ターミナル風デザイン** - エンジニア心をくすぐるCLI風インターフェース
- 💻 **コードスニペット** - JavaScript風の年賀メッセージ
- 🎮 **コナミコマンド** - `↑↑↓↓←→←→BA` で隠しモード発動！
- 🌸 **アニメーション** - 浮遊するエモジと紙吹雪
- 📱 **レスポンシブ** - スマホでも見やすいデザイン
- 🎵 **サウンドエフェクト** - コナミコマンド時にチャイム音

## 🚀 デプロイ方法

### GitHub Pages

1. このリポジトリをGitHubにプッシュ
2. Settings → Pages → Source: `main` branch を選択
3. 数分後に `https://<username>.github.io/<repo-name>/` で公開！

### Vercel

1. [Vercel](https://vercel.com) にログイン
2. 「New Project」から GitHubリポジトリをインポート
3. そのままデプロイ（設定不要！）

### Netlify

1. [Netlify](https://netlify.com) にログイン
2. 「Add new site」→「Import an existing project」
3. GitHubリポジトリを選択してデプロイ

## 🎮 隠し要素

### コナミコマンド
キーボードで `↑ ↑ ↓ ↓ ← → ← → B A` を入力すると...
- 🌈 レインボーエフェクト
- 🏇 走る馬のアニメーション
- 🎊 紙吹雪
- 🎵 チャイム音
- 🏆 シークレットメッセージ

### その他
- ターミナルをクリックでアニメーションスキップ
- 右下の 🎮 ボタンでも隠しモード発動
- コンソールにも特別メッセージ！

## 🛠️ カスタマイズ

### メッセージを変更
`index.html` の `.message-box` セクションを編集

### 色を変更
`style.css` の `:root` セクションで CSS変数を編集

### アニメーション速度
`script.js` の `steps` 配列で `delay` 値を調整
